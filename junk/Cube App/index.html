How to Use

  1. Use rotation buttons to orient the cube
  2. Use face rotation buttons to turn cube sides
  3. Click "Solve Cube" to generate a solution
  4. Follow the steps in the solving guide
  5. Use "Reset Cube" to return to initial state
  6. On touch devices, drag to rotate the cube view
